In data we thrust

Le simple fait de naviguer sur le Web génère de l'information. BEAUCOUP d'information.

Data = #hearth;

L'exploitation de ces données peut être positive et intéressante quand elle est réalisée de manière directe, explicite, pour apporter de la valeur ajoutée à l'utilisateur. Voici quelques exemples. - Suggestions d'achats par amazon (n'oublie pas d'acheter les piles qui vont avec cet appareil). - découverte de musique sur Youtube, Spotify, etc. - suggestions de films (Netflix).

Data = #hearth;

L'exploitation indirecte et globale peut également être bénéfique. - Google Flu permet de suivre et combattre les épidémies de grippe.
